# AIO_Control_Modular

Modular application to control and synchronize:
- Oscilloscope
- Waveform Generator
- Optical Spectrum Analyzer (OSA)

## Structure
- `controllers`: Device communication
- `gui`: User interfaces (Tkinter)
- `models`: Data management
- `utils`: Helper functions
